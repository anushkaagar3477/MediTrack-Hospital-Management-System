# ClinicFlow

ClinicFlow is a reduced full-stack web development project built with Flask, Jinja templates, JavaScript, and SQLite.

## Scope

This version keeps only the core web-app features:

- patient registration
- doctor directory
- appointment booking
- appointment and patient deletion

## Why this version

The original project felt closer to a DBMS or hospital ERP because it included AI modules, inventory tracking, reports, alerts, and profile/settings utilities. This reduced version focuses on a cleaner full-stack demo with:

- frontend forms and dynamic slot loading
- backend routes and APIs
- database persistence with SQLAlchemy
- end-to-end CRUD workflow
